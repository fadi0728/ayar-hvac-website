# Ayar Heating & Cooling Website

## Preview locally
1. Install Node.js
2. Run: npm install
3. Run: npm run dev

## Deploy to Vercel
1. Go to vercel.com
2. Create an account or sign in
3. Click Add New Project
4. Upload this folder or connect it through GitHub
5. Framework preset: Vite
6. Build command: npm run build
7. Output directory: dist
8. Deploy

After deployment, Vercel will give you a preview link.
Then add your domain ayarhvac.com under Project Settings > Domains.
